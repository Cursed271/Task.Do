# Task.Do

Task.Do is a to-do list written in python that helps you organize your tasks.

## Installation

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd Task.Do
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start Task.Do

```bash
  python Task.Do.py
```
